上映日期：2018年9月14日 日系纯爱大作，改编自动画片《我想吃掉你的胰脏》。 电影以回忆展开，男主知道了女同学患上胰脏癌的秘密，陪她度过了生命中最后的时光。 情节虽然催泪，但还是有值得开心的地方。 那就是，小栗旬又帅回来了！ ◆◆《江湖儿女》◆ ◆ 主演：廖凡 / 赵涛 / 徐峥 上映日期：2018年9月21日 国产文艺片的扛把子贾樟柯总算是回来了。 他的野心这次可是全用在《江湖儿女》上了。 一部电影串联起了之前的所有作品，打造出了一整部小城市变迁史。 三大廖凡、赵涛，徐峥全是连眼神都有戏的好演员。 不过文艺题材注定这部电影不会有太多排片，想看的一定要提前做好准备。 ◆◆《黄金兄弟》◆ ◆ 主演：郑伊健/ 陈小春 / 谢天华 / 钱嘉乐 / 林晓峰 上映日期：2018年9月21日 不一定好看，但一定会去看的电影。 《煎饼侠》里的匆匆一瞥根本满足不了《古惑仔》粉丝的想念。 这次五人众再次集齐，不如江湖，反而劫富济贫。 女主还是佘诗曼，没看够《延禧攻略》的可以再去瞅瞅娴妃啦。 ◆◆《李茶的姑妈》◆ ◆ 主演： 黄才伦 / 艾伦 / 宋阳 / 卢靖姗 / 沈腾 上映日期：2018年9月30日 “夏洛特”后，“李茶的”终于闪亮登场。 电影就是一个围绕有钱寡妇的喜剧，虽然初步看起来很单调，但这可是改编自开心麻花的同名爆款话剧，剧本肯定没毛病。 不过会不会像《夏洛特烦恼》一样大爆，还是得等上映了才好说。 ◆◆《影》◆ ◆ 主演：邓超 / 孙俪 / 郑凯 / 王千源 / 关晓彤 上映日期：2018年9月30日 张艺谋的电影一直都很“好看”。 他的配色从来就没让人失望过，每一帧都是直接能当壁纸的节奏。 《影》这次放弃了大红大绿，黑白灰的画面给人一种泼墨画的感觉，中国风十足。 而且邓超孙俪两口子这回终于正儿八经的合作了一部电影，想想都让人激动。 邓超一人分饰两角也是一大看点，《烈日灼心》后他演技能不能再突破一下，扒叔还是相当期待的。 不过看到演员表上的郑凯、关晓彤，还是希望娘娘能多带带他们吧。 ◆◆《悲伤逆流成河》◆ ◆ 主演：赵英博 / 任敏 / 辛云来 / 章若楠 / 朱丹妮 上映日期：2018年9月30日 不是郭！敬！明！导！演！ 本片是由出版过《胜者为王》的作家落落执导。 从选角来看没什么圈钱的痕迹，针对校园欺凌题材的青春片的确很少见。 预告很抓人，总感觉上映后还有更大的动静。 ◆◆《无双》◆ ◆ 主演：周润发 / 郭富城 / 张静初 上映日期：2018年9月30日 被《欧洲攻略》恶心到了？ 放心，《无双》绝对是一部地道的港片，光是发哥和郭富城的双主演就让人觉得靠谱。 电影是一场围绕国际假钞集团展开的双雄对决，虽然老套，但这几年还真不多见。 导演还致敬了一下小马哥的钞票点烟。 爱看港片的，别犹豫了。 ◆◆《胖子行动队》◆ ◆ 主演：文章 / 包贝尔 / 郭京飞 上映日期：2018年9月30日 文章和包贝尔继《陆垚知马俐》后的又一次合作。 看了预告怎么说呢。。。 这是一部非常适合带小朋友去看的电影。 ◆◆《找到你》◆ ◆ 主演：姚晨 / 马伊琍 上映日期：2018年9月30日 又是一部没多少钱做宣传的电影。
